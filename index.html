PhysioNet 2019 | PyTorch, Transformers, Time-Series ML

Key Features

  • Designed and implemented a Dynamic Tanh (DyT) Transformer for early sepsis prediction on 1.55M ICU time steps from 40K patients.
  • Replaced LayerNorm with adaptive DyT normalization, preserving clinically meaningful physiological spikes and improving robustness.
  • Developed temporal attention by injecting time-gap embeddings into self-attention to model irregular ICU sampling intervals.
  • Engineered a missingness-aware preprocessing pipeline (forward fill + binary masks), treating observation patterns as predictive signals.
  • Trained using Focal Binary Cross-Entropy to handle ~1.8% positive labels, significantly improving early-warning precision.
  • Achieved AUROC 0.90 and ~3× AUPRC improvement over a LayerNorm Transformer baseline.
  • Built a Streamlit-based real-time monitoring dashboard to visualize patient risk trajectories.
